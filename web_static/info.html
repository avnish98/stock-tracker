How it works?

Stock tracker is a web application for fetching data of a NASDAQ listed company using ticker symbol. It uses iexfinance API for fetching data and Dash, a data visualistion framework by plot.ly .

Enter any ticker symbol for a NASDAQ listed company (such as "AAPL","GOOGL","MSFT") and data displays in realtime.

How I made it?

Data Collection

I used iexfinance API for stock data. It provides per day data including Open price, Close price,High & Low values and Volume of the stock. The fetched dataset look like.

data1 data1 data1 data1

Exploratory Data Analysis

From the data provided by API(open, close, high, low and volume), I calculated Percentage change using formula.

Percentage Change = ((Current Price - Open Price)/Open Price)x100

I also used a "ticker.csv" file for fetching name of the company from its ticker symbol.

Data Visualisation

I used Dash for Data Visualisation which is a summation of Flask Framework, Plot.ly and ReactJS, for real time update of graph and data.
